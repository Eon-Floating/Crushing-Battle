# 肠碎大战数字版

这是一个零依赖的双人联机原型：只需要 Node.js，就能在浏览器里创建房间、加入房间并游玩。

## 运行

```powershell
npm start
```

然后打开：

```text
http://localhost:3000
```

一个玩家点击“创建房间”，另一个玩家在新浏览器窗口或另一台同一局域网电脑中输入房间号加入。

## 当前规则实现

- 7x7 地图，右下角为终点，不可铺肠碎，共 48 个可用格。
- 铺肠碎阶段：抛硬币决定先手，每次沿蛇形路线铺 1 到 6 个，不能跳格。
- 出牌阶段：两名玩家使用同一个公共牌库，初始各 3 张手牌，每回合先抽 1 张再出 1 张。
- 营养和垃圾分为初级、中级、高级，分别填 1/2/3 个尚未填满的肠碎。
- 已方肠碎被营养填满加分，被垃圾填满扣分。
- 夺取、快手还击、翻面、窜、超绝消化、超弱消化都已做成可用技能。
- 最后一张资源牌打出后结算；没有可出牌时会自动跳过当前回合。

## 说明

这是为了学习开发而做的“简单但能玩”的第一版。服务器把房间存在内存里，所以关掉命令行窗口后房间会消失。之后可以继续加账号、重连、观战、动画、部署公网服务器等功能。
